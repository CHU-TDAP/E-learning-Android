客戶端界面應用程式
===
需搭配採用[E-learning-Server](https://github.com/CHU-TDAP/E-learning-Server)後端伺服器使用

## 簡介
提供給學生在Android裝置上的操作界面，讓學生能在博物館內利用此應用程式進行學習導引，除了提供教材外，要提供給學生下一個學習點。

## 需求
- 預設採用版本為: Android 4.2.2 (API Lev 17)
- Wifi or 3G <- 有網路即可
- 相機鏡頭（掃描QR Code時需要）
